# Expense Tracker (Python)

A simple command-line Expense Tracker built with Python that helps users record, categorize, and analyze their daily expenses. The application stores expense data in a CSV file and provides summaries based on categories and total spending.

## Features

* Add new expenses
* Categorize expenses
* Automatically record the current date
* View all expenses
* Calculate total expenses
* Display expenses by category
* Store data permanently using CSV files
* Input validation for expense amounts

## Technologies Used

* Python
* CSV File Handling
* Functions
* Lists and Dictionaries
* Date and Time Module

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/expense-tracker-python.git
```

2. Navigate to the project directory:

```bash
cd expense-tracker-python
```

3. Run the application:

```bash
python main.py
```
